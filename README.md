# Britlex Landing Page

A responsive landing page built from a Figma design using HTML and CSS.

## About

This project is a practice implementation of a landing page based on a design mockup. The main goal was to improve HTML and CSS layout skills, including responsive design and semantic markup.

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design

## Features

- Semantic HTML
- Responsive layout
- Accessibility basics (alt attributes, labels, semantic elements)
- BEM methodology
